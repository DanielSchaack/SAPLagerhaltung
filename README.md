# SAPLagerhaltung
rudimentäre SAPUI5-Anwendung für eine Lagerhaltung. Work in Progress.

Das Projekt entsteht in Zusammenarbeit mit Patrick Dröge für das Fach "ERP-Systeme".
Der Stand vom 12.07.2020 ist noch nicht der finale Zustand. Das Projekt wird weiterhin bearbeitet. Es fehlen noch Funktionen wie eine Funktionierende Such-Funktion sowie eine Detail-View der vorhandenenProdukte.

Es besteht aus 2 Parts;
dem Lager, welches die Grundlage für die Datenbank vom oData-Service bildet,
und dem Supermarkt, welches das Interface zur Interaktion darstellt.
